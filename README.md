## Description:

CMS style app using node and mysql as backend. Features consist in adding/managing employees database easily

<a name="installation"></a>

## Installation:

    npm install

<a name="usage"></a>

## Usage:

Start Mysql server and run

```
source db/db.sql
source db/schema.sql
source db/seeds.sql
```

Then Run `node index.js`.
